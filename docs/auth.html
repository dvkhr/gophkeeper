Overview ▾

Package auth предоставляет функции аутентификации и авторизации для GophKeeper. Включает: - генерацию и парсинг JWT-токенов, - работу с refresh-токенами, - хэширование и проверку паролей, - контекстный интерсептор для gRPC.

func AuthInterceptor ¶ 

func AuthInterceptor(cfg config.Config, repo repository.TokenRepository) grpc.UnaryServerInterceptor

AuthInterceptor — gRPC middleware для проверки JWT-токена в заголовках. Пропускает методы /keeper.KeeperService/Login и /keeper.KeeperService/Register без проверки. Для остальных методов: - извлекает Bearer-токен, - проверяет его валидность, - проверяет, не отозван ли он, - добавляет userID в контекст.

func CheckPasswordHash ¶ 

func CheckPasswordHash(password, hash string) bool

CheckPasswordHash проверяет, совпадает ли пароль с хэшем. Возвращает true, если пароль совпадает, и false в противном случае.

func GenerateRandomString(n int) string

GenerateRandomString генерирует случайную строку указанной длины. Используется для создания refresh-токенов.

func GenerateRefreshToken ¶ 

func GenerateRefreshToken(repo repository.TokenRepository, userID string, cfg config.Config) (string, error)

GenerateRefreshToken генерирует случайный refresh-токен и сохраняет его в БД. Токен действителен в течение RefreshTokenTTLDays дней.

type Claims ¶

Claims — структура полезной нагрузки (payload) JWT-токена. Включает идентификатор пользователя и стандартные claims (ExpiresAt, IssuedAt, Issuer и др.). 

type Claims struct {
    UserID string `json:"user_id"`
    jwt.RegisteredClaims
}

func ParseToken ¶ 

func ParseToken(cfg config.Config, tokenStr string) (*Claims, error)

ParseToken — разбирает строку токена и возвращает claims. Возвращает ошибку, если токен недействителен или подпись не совпадает.
